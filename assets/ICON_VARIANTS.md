## DexDictate Icon Variants

### File Groups

- `dexdictate-icon-standard-01.png` through `dexdictate-icon-standard-11.png`
  Standard DexDictate logo variants. These are the clean blue roundel versions with the dog and microphone only.
- `dexdictate-icon-standard-sheet-01.png` and `dexdictate-icon-standard-sheet-02.png`
  Source sheets containing three standard logo variants each. These are raw composite inputs, not direct app icon outputs.
- `dexdictate-icon-canada-01.png` through `dexdictate-icon-canada-05.png`
  Canada variants. These add explicit Canada-specific elements such as maple leaves, Canadian flags, red-and-white palettes, winter gear, and regional scene props.
- `dexdictate-icon-aussie-01.png` and `dexdictate-icon-aussie-02.png`
  Australia variants. These use Aussie labeling and Australian scene elements.

### Difference Between Canada And Standard

- `Standard` versions are the neutral brand set. They keep the core dog-plus-microphone mark in a simple blue circular badge without country-specific props.
- `Canada` versions are themed variants. They keep the same core character but layer in maple-leaf iconography, Canadian flags, red-and-white treatment, winter gear, and more detailed environmental decoration.
- `Standard` reads as the default product identity. `Canada` reads as a regional or campaign variant.

### New Standard Splits

- `dexdictate-icon-standard-06.png` through `dexdictate-icon-standard-11.png`
  Split from the two standard source sheets. Each one was isolated from a three-up composite, converted to transparent background, and normalized to the same `1024x1024` transparent canvas format as the rest of the icon set.
